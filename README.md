 Smart Study Planner & Performance Tracker

📌 Description

A web-based application that helps students manage their academic activities efficiently. It allows users to organize timetables, track subjects, store syllabus, and generate weekly study plans.

 🚀 Features

* Timetable management
* Add and manage subjects
* Store syllabus
* Weekly study plan generation
* Data storage using SQL

 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python
* Database: SQL

▶️ How to Run

1. Clone the repository
2. Run backend (Python)
3. Open frontend in browser

 📈 Future Improvements

* User login system
* Progress tracking dashboard
* Mobile responsiveness
